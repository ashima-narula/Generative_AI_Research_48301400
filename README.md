# Generative AI Research

## Project Overview
This repository stores materials for the research project **Using Generative AI Tools – Boon or Bane**.  
It contains research data, analysis scripts, reports, and supporting materials developed during the project.

## Repository Structure

### data/
Contains research data and analysis materials, including:
- Survey datasets (CSV, Excel)
- Interview transcripts
- Survey questions
- Consent forms
- Data visualisations

### scripts/
Contains code and analysis files, including:
- Python scripts
- Data cleaning scripts
- Quantitative analysis files

### sources/
Contains literature review resources:
- Journal articles
- Conference papers
- Books
- News articles
- Reference notes

### reports/
Contains project documentation:
- Draft proposals
- Analysis reports
- Conference papers
- Final research reports

### media/
Contains supporting materials:
- Images
- Photos
- Information sheets
- Other media files

## Collaboration Guidelines
- Use clear file naming conventions (example: YYYYMMDD_filename_version)
- Commit changes with meaningful messages
- Use branches for major updates before merging
- Update documentation when adding or changing files
- Review version history regularly through GitHub Activity

## Access Control
### Full Access
Shared among research team members for:
- Literature review materials
- Scripts and code
- Draft reports
- Non-sensitive visualisations

### Restricted Access
Limited to authorised researchers for:
- Raw survey responses
- Interview transcripts
- Consent forms
- Participant information
- Sensitive unpublished analysis data

Sensitive files should be protected using permissions, passwords, and secure storage.

## Version Control
This repository uses Git and GitHub to:
- Track project development
- Manage revisions
- Support collaboration
- Maintain reproducibility
